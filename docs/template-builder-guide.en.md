# Template Builder User Guide

The Jamail Template Builder allows you to create email templates easily using a drag-and-drop interface without any coding knowledge.

## 1. Accessing the Builder

1. Select a template to edit from the template list or create a new one.
2. Click the **"üé® Builder"** button in the top right corner of the template detail page.
   - *Note: You can switch to the Builder from a 'Basic' type template, but existing HTML content may not be converted to the builder-compatible format.*

## 2. Interface Overview

The builder interface is divided into three main sections:

1.  **Sidebar (Left)**: Contains the list of available blocks.
2.  **Canvas (Center)**: The workspace where you place blocks and preview the email layout.
3.  **Properties Panel (Right)**: Where you edit the details (content, styles, etc.) of the selected block.

## 3. Basic Usage

### Adding Blocks
Click and drag a block from the left sidebar and drop it onto the center canvas.

### Editing Blocks
Click on a block placed on the canvas to activate the Properties Panel on the right. Here you can modify text content, image URLs, colors, and more.

### Reordering Blocks
Drag blocks within the canvas to change their vertical order.

### Deleting Blocks
Select the block you want to remove, then click the **"Delete Block"** button at the bottom of the Properties Panel.

## 5. Saving and Exiting

When you are finished, click the **"Save"** button in the top right corner to save your changes. Once saved, you can return to the template detail page using your browser's back button.
