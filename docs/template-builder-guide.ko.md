# 템플릿 빌더 사용 가이드

Jamail 템플릿 빌더는 코딩 지식 없이도 드래그 앤 드롭 방식으로 이메일 템플릿을 쉽게 제작할 수 있는 도구입니다.

## 1. 빌더 접속 방법

1. 템플릿 목록에서 편집할 템플릿을 선택하거나 새 템플릿을 생성합니다.
2. 템플릿 상세 페이지 우측 상단의 **"🎨 Builder"** 버튼을 클릭합니다.
   - *참고: 'Basic' 타입 템플릿에서도 빌더로 전환할 수 있으나, 기존 HTML 내용은 빌더 호환 형식으로 변환되지 않을 수 있습니다.*

## 2. 화면 구성

빌더 화면은 크게 세 부분으로 나뉩니다:

1.  **사이드바 (왼쪽)**: 사용 가능한 블록 목록이 있습니다.
2.  **캔버스 (중앙)**: 블록을 배치하고 이메일의 실제 모습을 미리 보는 작업 공간입니다.
3.  **속성 패널 (오른쪽)**: 선택한 블록의 세부 설정(내용, 스타일 등)을 편집하는 곳입니다.

## 3. 기본 사용법

### 블록 추가하기
왼쪽 사이드바에서 원하는 블록을 클릭한 상태로 중앙 캔버스로 드래그하여 놓습니다.

### 블록 편집하기
캔버스에 배치된 블록을 클릭하면 오른쪽 속성 패널이 활성화됩니다. 여기서 텍스트 내용, 이미지 주소, 색상 등을 수정할 수 있습니다.

### 블록 순서 변경
캔버스 내의 블록을 드래그하여 위아래로 순서를 변경할 수 있습니다.

### 블록 삭제
삭제하려는 블록을 선택한 후, 속성 패널 하단의 **"Delete Block"** 버튼을 클릭합니다.

## 4. 블록 종류 및 속성

### 📝 텍스트 (Text)
일반적인 문단을 작성할 때 사용합니다.
- **Content**: 텍스트 내용을 입력합니다.
- **Styles**: 글자 크기, 색상, 배경색, 여백 등을 조절할 수 있습니다.

### 🖼️ 이미지 (Image)
외부 이미지를 삽입할 때 사용합니다.
- **Image URL**: 이미지의 주소(URL)를 입력합니다.
- **Alt Text**: 이미지가 로드되지 않을 때 표시될 대체 텍스트를 입력합니다.
- **Styles**: 이미지 주변의 여백 등을 조절할 수 있습니다.

### 🔘 버튼 (Button)
클릭 가능한 링크 버튼을 만듭니다.
- **Content**: 버튼에 표시될 텍스트를 입력합니다.
- **Link URL**: 버튼 클릭 시 이동할 웹페이지 주소를 입력합니다.
- **Styles**: 버튼 색상, 글자 색상, 크기 등을 조절하여 디자인할 수 있습니다.

### ➖ 구분선 (Divider)
내용 사이에 수평선을 넣어 구획을 나눕니다.
- **Styles**: 구분선의 여백, 배경색 등을 조절할 수 있습니다.

### 💻 HTML
직접 HTML 코드를 삽입해야 할 때 사용합니다. 복잡한 레이아웃이나 커스텀 디자인이 필요할 때 유용합니다.
- **HTML Code**: 사용할 HTML 코드를 직접 입력합니다.

## 5. 저장 및 나가기

작업이 완료되면 우측 상단의 **"Save"** 버튼을 눌러 변경 사항을 저장합니다. 저장이 완료되면 브라우저 뒤로 가기 버튼을 통해 템플릿 상세 페이지로 돌아갈 수 있습니다.
